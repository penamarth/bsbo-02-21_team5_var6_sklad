```plantuml
@startuml
actor Кладовщик
entity "Управляющая компания" as УК
entity "Система сканирования товара" as ССТ
database "Система учета" as СУ

== Инициация поставки товара ==
УК -> Кладовщик : Отправляет товар и документы
УК -> СУ : Регистрирует отправку товара

== Приемка товара кладовщиком ==
Кладовщик -> СУ : Проверяет и сопоставляет товар с документами

== Сканирование товара ==
Кладовщик -> ССТ : Сканирует каждый товар
ССТ -> СУ : Отправляет данные о товаре (название, количество, дата поступления)

== Регистрация товара в системе учета ==
СУ -> СУ : Обновляет базу данных, регистрирует новый товар

== Подтверждение приемки товара ==
Кладовщик -> СУ : Подтверждает корректность регистрации

== Уведомление управляющей компании ==
СУ -> УК : Отправляет уведомление о приемке товара
@enduml
